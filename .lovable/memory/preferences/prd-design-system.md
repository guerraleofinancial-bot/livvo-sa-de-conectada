# PRD Master — Volume 6: Design System e UX

## Identidade
Visual clean, HealthTech moderna, consistência entre módulos.

## Paleta
- **Primary teal** (#14B8A6).
- Neutros claros.
- Sucesso / alerta / erro padronizados.

## Componentes (obrigatórios do DS)
Botões, cards, inputs, modais, tabelas, badges, gráficos.

## UX
- Menos cliques.
- Feedback imediato.
- Responsividade.
- Acessibilidade WCAG AA.

## Responsividade
Desktop, tablet, mobile.

## Padrões
Ícones consistentes, espaçamentos, tipografia, estados de carregamento, estados vazios.

## Critérios de Aceite
- Nenhum componente fora do Design System.
- Nova funcionalidade deve reutilizar componentes existentes.

## Objetivo
Ser o principal marketplace de saúde presencial do Brasil. Maximizar conversão visitante→agendamento. Crescimento sustentável do parceiro.

## Jornada Paciente
Buscar (especialidade/procedimento/sintoma/cidade/convênio) → comparar → perfil completo → agendar em poucos cliques → cobrar pela Livvo → pagar → histórico.

## Jornada Profissional
Criar perfil → docs → aprovação → agenda+serviços → receber pacientes → CRM+cobranças → métricas.

## Perfil Público Premium
Banner + foto + vídeo · Selo Parceiro Verificado · serviços com preço inicial e duração · formação/especializações/experiência · avaliações verificadas · CTAs: **Agendar · WhatsApp · Ligar · Compartilhar**.

## Busca Inteligente
Eixos: especialidade, sintomas (taxonomia futura), procedimento, convênio, cidade/bairro.
Filtros: disponibilidade, preço, avaliação.

## Ranking (search_providers_ranked)
Prioridade em ordem:
1. Parceiro verificado
2. Completude do perfil
3. Tempo de resposta
4. Avaliações
5. Disponibilidade
Patrocinados **sempre visualmente separados** dos orgânicos.

## Monetização
Free · Premium · Destaques patrocinados · Comissão sobre consultas (regras vigentes) · Programa Parceiros Fundadores.

## Critérios de aceite (invioláveis)
- Agendamento em ≤ 3 etapas.
- Nenhum profissional não aprovado aparece publicamente.
- Perfil responsivo desktop + mobile.
- SEO por perfil (title/description/og únicos por rota).
- Link compartilhável para cada parceiro (`/app/profissional/:id`, `/app/empresa/:id`).
